NoSQL Cleaner

NoSQL Cleaner is a tool that allows a user to inspect and clean/correct data that is to be inserted in a MongoDB or Neo4j database. The primary goal of this tool is to provide an interface with which someone can inspect the values and fields of a document to be inserted in a database and correct any incorrect data as necessary.
